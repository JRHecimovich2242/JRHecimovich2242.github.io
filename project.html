Stormlight Archive 2D Platformer

Made with Unity. November 2019 - Present.

Individual Project

As a first engagement with designing and developing my own game, I chose to adapt some aspects of Brandon Sanderson's The Stormlight Archive and create a 2D platformer demo utilizing features of the magic system described in the novels.

Features

Dynamic Gravity System

Using the WASD keys, the player can change the direction in which gravity pulls them. This is accomplished by manipulating Unity Rigidbody2D and ConstantForce2D components. In addition, animations are adjusted to display properly no matter the player's orientation.

Mana and "Stormlight"

Gravity changing (or "Lashing") is powered by mana ("Stormlight") which can be obtained both through defeating enemies and by engaging with interactable objects found throughout the levels. This stormlight can be used to interact with the environment, to lash, or even to restore lost health.

Combat

Currently this demo features a single brawler enemy archetype, but more enemy archetypes are planned. These enemies can be defeated through traditional means, or by lashing various interactable objects towards them. When stunned, the enemies themselves can be lashed around the environment, but all at the cost of stormlight.
